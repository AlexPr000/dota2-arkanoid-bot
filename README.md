## Инструкция по настройке и эксплуатации

1. Запустите клиент Dota 2 на целевом мониторе и перейдите в интерфейс мини-игры.
2. Запустите исполняемый файл `Dota2_Arkanoid_Bot.exe` или скрипт `dota_bot.py`.
3. Введите индекс монитора, на котором развернута игра, в поле ввода.
4. Нажмите кнопку "1. ТЕСТ РАМОК" для включения режима калибровки. Настройте ползунки:
   * **Левая/Правая граница X, Верх поля Y:** Ограничивают рабочую зону.
   * **Разделитель Y:** Отделяет зону полета сапога от траектории вагонетки. Должен находиться строго над крышей кареты.
   * **Низ поля Y:** Ограничивает нижнюю границу видимости вагонетки, ограничитель ставить по верхну.
5. Нажмите кнопку "СОХРАНИТЬ КОНФИГУРАЦИЮ" для записи параметров в файл настроек JSON.
6. Снимите флаг "Превью дебага" для прекращения рендеринга окна OpenCV и повышения частоты обработки кадров.
7. Нажмите кнопку "2. АВТОПИЛОТ" для перевода программы в рабочий режим.

## Используемый стек технологий

* **Язык программирования:** Python 3.11
* **Библиотеки:** OpenCV-Python (компьютерное зрение), mss (скриншоты), tkinter (графический интерфейс), pydirectinput (ввод DirectX).

## Ложные срабатывания антивирусного ПО

Скомпилированный файл проекта (.exe) может вызывать ложные предупреждения со стороны встроенных средств защиты Windows Defender или сторонних антивирусных программ. Это обусловлено спецификой работы компилятора PyInstaller и использованием низкоуровневых библиотек перехвата и эмуляции ввода (pydirectinput). Исходный код является полностью открытым и доступен для аудита в данном репозитории. При необходимости добавьте исполняемый файл в список исключений вашей системы защиты.
